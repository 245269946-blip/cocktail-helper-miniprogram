# 调酒助手微信小程序

定位：年轻人饮品决策助手，帮助用户快速决定今晚喝什么。

## 第二版方向

- 首页改为行动入口优先：去便利店买、家里有什么、找一杯好喝的、第一次尝试
- 搜索支持鸡尾酒名、基酒、材料、口味、便利店等明确词，并提供分层联想
- 结果页强化适合谁、风味评分、执行信息和继续探索
- 详情页先建立想喝感，再展示材料、替代方案、便利店版和家里简化版
- 家里有什么按“现在直接能做、只差 1 样、便利店补一下、升级版更好喝”分层
- 便利店模式改成套餐卡，突出预算、杯数、成功率和购物清单感

## 当前内容

- 原生微信小程序框架
- 35 个配方、8 个便利店/场景方案、11 类基酒、13 类常见材料
- 搜索、推荐、收藏、缺料判断、云开发数据导入
- 云端优先、本地数据兜底

## 运行方式

1. 打开微信开发者工具
2. 选择“导入项目”
3. 项目目录选择本目录
4. AppID 可使用测试号或继续使用 `touristappid`

## 云开发

如需使用微信云数据库，请参考 `CLOUD.md`，并导入 `cloud-data/` 下的 JSON 数据。
