# 调酒助手微信小程序

定位：鸡尾酒配方搜索 + 风味推荐 + 场景购买指南。

当前版本已完成：

- 原生微信小程序框架
- 首页搜索型入口
- 搜配方、按基酒找、按材料找、便利店调酒、家里有什么、新手推荐
- 风味分类、场景推荐、搜索结果页与标准详情页
- 35 个配方、8 个场景方案、11 类基酒、13 类常见材料
- 微信云开发接入：云端优先，本地兜底

## 微信云开发

源码包内包含 `CLOUD.md` 和 `cloud-data/`。

需要创建云数据库集合：

```text
recipes
schemes
bases
ingredients
appContent
```

然后导入 `cloud-data` 目录下的 JSON 文件。
