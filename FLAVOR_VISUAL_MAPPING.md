# 酒类视觉映射系统

这个文件用于固定不同酒类、场景和推荐状态的视觉语言。后续所有页面、插画和动效都必须从这里取规则，不能每页重新发明风格。

统一底层：奶油纸感、轻颗粒、透明空气感、柔和边缘、低饱和色、少量青绿色品牌色。

差异来自口味视觉，而不是页面结构。

## 金酒 Gin

关键词：青柠、冰感、气泡、透明、夏夜、青绿色冷光。

主色：青绿色、冰蓝、青柠黄绿。

材质：透明玻璃、气泡水、冷凝水、清透冰块。

插画元素：高球杯、青柠圆片、细气泡、透明液体。

适用页面：金酒兑什么、金汤力、金酒雪碧、清爽酒单。

禁止：太厚重、太酒吧、黑金专业感。

## 威士忌 Whisky

关键词：琥珀、木桌、夜晚、冰球、暖光、微醺。

主色：琥珀、暖棕、柚子黄、少量青绿色品牌点缀。

材质：厚底玻璃、冰球、轻木纹、温暖灯光。

插画元素：Old fashioned 玻璃杯、大冰球、琥珀色液体、小瓶标签。

适用页面：威士忌兑什么、威士忌可乐、Highball、聚会低门槛。

禁止：传统男士威士忌广告、黑金奢华风、过度烟熏和暗色。

## 百利甜 Baileys

关键词：奶香、咖啡、巧克力、丝滑、甜品感。

主色：奶油白、拿铁棕、巧克力浅棕、少量青绿色品牌点缀。

材质：牛奶旋涡、咖啡泡沫、柔软布料、甜品桌面。

插画元素：奶油色液体、咖啡杯或矮杯、巧克力碎片、轻泡沫。

适用页面：百利甜怎么喝、百利甜牛奶、百利甜咖啡、甜口入门。

禁止：变成奶茶小程序、过度甜品店、颜色过棕过脏。

## 伏特加 Vodka

关键词：透明、冷蓝、冰块、干净、柠檬。

主色：冷蓝、透明白、柠檬黄、少量青绿色品牌点缀。

材质：冷玻璃、大冰块、透明液体、极简冷光。

插画元素：透明杯、柠檬角、冰块、简洁瓶身。

适用页面：伏特加兑什么、伏特加苏打、伏特加橙汁、低糖低酒感。

禁止：夜店蓝紫、过度冷硬科技感。

## 朗姆 Rum

关键词：热带水果、薄荷、橙黄、夏天、酸甜。

主色：橙黄、薄荷绿、蜜桃、少量青绿色品牌点缀。

材质：水果切片、薄荷叶、碎冰、夏天光感。

插画元素：莫吉托杯、薄荷叶、青柠、热带水果。

适用页面：朗姆兑什么、莫吉托、代基里、酸甜清爽。

禁止：过度海岛度假风、饱和热带海报。

## 便利店调酒

关键词：冰柜、夜晚便利店、小瓶饮料、冷光、即时满足。

主色：冰柜蓝、青绿色、奶油白、少量柚子黄。

材质：塑料瓶、冰杯、小票贴纸、冰柜反光。

插画元素：冰杯、小瓶酒、饮料瓶、便利店标签、冷柜光带。

适用页面：便利店调酒、购买方案、临时采购。

禁止：电商货架感、真实便利店照片堆叠、商品网格。

## 新手友好

关键词：安全、干净、容易成功、不翻车。

主色：明亮奶油、浅青绿、低对比。

材质：留白、简单杯型、少量冰块。

插画元素：简单杯子、1-2 个材料、清晰步骤符号。

视觉规则：留白更多、标签更少、CTA 更明确。

## 更低酒感

关键词：轻、柔和、不刺激、像饮料。

主色：半透明浅色、水果色、轻气泡。

材质：水果汁感、低透明度液体、轻气泡。

插画元素：水果切片、浅色液体、细气泡、小杯或高球杯。

视觉规则：酒瓶弱化、饮料感增强、颜色更轻。

## 映射优先级

当一个配方同时命中多个规则时，按以下顺序决定视觉主题：

1. 具体基酒：金酒 / 威士忌 / 百利甜 / 伏特加 / 朗姆
2. 场景：便利店调酒
3. 状态：新手友好 / 更低酒感
4. 风味：甜口 / 清爽 / 奶香 / 茶感 / 咖啡感

例如：金酒便利店方案主视觉仍为金酒，局部加入便利店元素；百利甜新手推荐主视觉仍为百利甜，版式更明亮更留白；便利店无酒精方案主视觉为便利店，酒瓶弱化。

## 统一限制

每个主题可以改变：主色、液体颜色、杯型、局部材质、辅助元素。

不能改变：页面骨架、按钮系统、标签系统、字体层级、奶油纸感和空气感、轻手绘水彩基调。
