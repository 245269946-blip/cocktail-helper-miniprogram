# 🍸 调酒助手微信小程序 — 项目概览

> **定位**：年轻人饮品决策助手，帮助用户快速决定今晚喝什么。  
> **技术栈**：原生微信小程序 + 微信云开发  
> **当前内容量**：35 个配方、8 个便利店/场景方案、11 类基酒、13 类常见材料

## 🎯 产品架构：三层体验设计

项目采用严格的**三层分层架构**，确保品牌情绪、工具效率和决策支持各司其职：

### Layer 1：品牌情绪启动层
- **职责**：让用户「想喝」
- **实现**：首页启动屏（夏夜便利店饮品实验室），CSS 绘制的冰块、气泡、冰柜灯光动效
- **核心动作**：一个按钮「开始今天第一杯」→ 进入 Layer 2
- **原则**：功能密度低，视觉沉浸，不堆功能入口

### Layer 2：真正首页
- **职责**：让用户快速找到喝法
- **核心功能**：
  - 🔍 搜索框（支持鸡尾酒名、基酒、材料、口味、便利店等关键词）
  - 🏷️ 热门搜索标签 + 最近搜索
  - 🚪 三大高频入口：家里有什么 / 便利店饮品 / 按材料找
  - 📋 快捷直达入口：便利店能买齐 / 无酒精也好喝 / 像冰饮一样清爽
- **原则**：搜索是第一操作，信息密度高但视觉克制

### Layer 3：搜索结果 / 配方决策层
- **职责**：让用户快速决定怎么喝
- **结果页**：主推荐卡 → 扩展喝法 → 完整酒单
- **详情页**：大图 → 风味评分（★☆）→ 材料/替代版 → 调制步骤 → 继续探索 → 替代方案 → 翻车提醒
- **原则**：第一眼先看哪杯，立刻能进入详情

## 🎨 视觉系统

- **奶油纸感**、轻颗粒、透明空气感、柔和边缘、低饱和色
- **品牌主色**：青绿色 `#2f948c`
- **背景色**：奶油暖白 `#f6f3ed`

| 伏特加 Vodka | `#3c9fc4` 冷蓝 | 透明、冰块、干净 |
| 朗姆 Rum | `#d8893a` 橙黄 | 热带水果、薄荷、夏天 |

### 插画资源（83 张水彩风 PNG）
- **饮品插画** 46 张：每个配方一张（金汤力、莫吉托、百利甜牛奶...）
- **基酒插画** 12 张：金酒、伏特加、威士忌、朗姆、龙舌兰、百利甜、梅酒、清酒等
- **材料插画** 25 张：可乐、雪碧、气泡水、橙汁、柠檬、薄荷、冰块等
- **风格**：水彩、轻颗粒、透明空气感、柔和边缘、微晕染

### UI 组件规范
- **主按钮**：青绿色渐变、胶囊圆角、轻阴影
- **次按钮**：奶油浅底、细描边
- **风味标签**：圆角胶囊、色彩跟随主题
- **功能标签**：半透明奶油底、不抢 CTA
- **特殊标签**：手写纸条感、轻旋转

### 动效
- 搜索框：默认毛玻璃 → 聚焦青绿发光
- 卡片：按下微沉 → 松开回弹
- 气泡：缓慢上浮、透明度变化
- 页面切换：淡入 + 轻微上滑

### 五维风味评分系统（drinkView.js）
| 维度 | 1 分 | 3 分 | 5 分 |
|------|------|------|------|
| 甜度 | 完全不甜 | 顺口 | 像甜品 |
| 酒感 | 无酒感 | 会慢慢有感觉 | 适合老手 |
| 清爽 | 厚重 | 还算清爽 | 很适合夏天 |
| 微醺 | 没负担 | 别喝太快 | 后劲明显 |
| 难度 | 倒一起就行 | 需要一点手法 | 进阶尝试 |

### 搜索与推荐（recommend.js）
- 支持：鸡尾酒名、基酒、材料、口味、便利店等关键词
- 匹配维度：名称、英文名、基酒、标签、材料名、场景
- 结果分层：主推荐 → 扩展喝法（更甜/更清爽/更低酒感/更像饮料）→ 完整酒单

## ☁️ 云开发

- **策略**：云端优先，本地兜底
- **数据同步**：进入页面时读取云数据库，云端不可用自动回退 `data.js`
- **集合**：recipes / schemes / bases / ingredients / appContent
- **导入**：`cloud-data/` 目录提供 JSON 格式初始数据
- **导出**：`node scripts/export-cloud-data.js` 可从本地重新生成云端数据

## 🔧 核心模块依赖关系

```
app.js
 └── contentStore.js ← data.js + recommend.js
      └── pages/* 
           ├── drinkView.js ← illustrations.js
           └── visualSystem.js
```

- **data.js**：纯数据层，所有配方/基酒/材料/入口配置
- **contentStore.js**：数据仓库，合并云端+本地数据
- **recommend.js**：搜索匹配、推荐排序、查找函数
- **drinkView.js**：视图组装，将数据转为 UI 所需格式
- **illustrations.js**：插画路径映射，统一资源引用
- **visualSystem.js**：主题匹配，根据配方特征自动选择视觉主题
